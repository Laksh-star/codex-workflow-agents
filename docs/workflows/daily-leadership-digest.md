# Daily Leadership Digest

## Buyer

Managers, directors, VPs

## Job to be done

Provide a clean daily view of progress, risks, blockers, decisions, and escalations across teams.

## Inputs

- Slack and Teams discussions
- GitHub or project delivery signals
- local status notes

## Workflow

1. Review the latest cross-team activity.
2. Extract notable wins, blockers, decisions, and unresolved risks.
3. Rank what leadership should care about today.
4. Draft a digest and optional owner tracker.
5. Schedule a recurring run for daily refresh.

## Deliverables

- leadership digest
- escalation list
- optional owner tracker workbook

## Differentiators

- cross-source synthesis
- recurring automations
- optional structured workbook for follow-ups

## Success metrics

- cuts daily status preparation time
- improves escalation quality
- reduces missed cross-team blockers
