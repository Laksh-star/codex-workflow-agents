# Executive Briefing Machine

## Buyer

Founders, business leads, chiefs of staff

## Job to be done

Compress scattered operating signals into a board-ready or leadership-ready packet without manual synthesis work.

## Inputs

- Slack and Teams threads
- GitHub updates when product delivery matters
- KPI files and local notes
- dashboard UIs through `computer-use`

## Workflow

1. Gather operating context from connected systems.
2. Pull or capture the latest KPI inputs.
3. Synthesize the narrative: wins, risks, blockers, and asks.
4. Generate an editable KPI workbook.
5. Generate an editable briefing deck.

## Deliverables

- KPI workbook
- leadership or board deck
- concise written briefing

## Differentiators

- native `Excel` and `PowerPoint` outputs
- automation for recurring refreshes
- `computer-use` for systems without APIs

## Success metrics

- saves 1 to 2 hours per cycle
- output is presentation-ready with light or no cleanup
- refreshes cleanly on a weekly cadence
